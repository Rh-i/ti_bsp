# 使用须知

此工程基于EIDE，以及keil的链接文件。由于未知原因，导出KEIL模板之后，在keil中无法编译。

syscfg使用的是bat文件自动打开。使用syscfg时，保存文件需要注意，保存到core中，而且要看好文件夹，不要保存错了。

syscfg需要根据自己电脑上的文件位置和版本修改，有两个位置有这个bat文件：eide文件夹中，和主文件夹。

已经配置好了CORTEX-DEBUG，可以实时显示变量值，使用的是JLink，稳定性高。
